## Slime lazy page
### Add attribute **slime-lazy-src**
>**example:**
```html
<img slime-lazy-src="source_images" />
<video>
  <source slime-lazy-src="video.mp4" />
</video>
```
### Support tag
- [x] style
- [x] img
- [x] video
- [x] iframe
- [x] embed
