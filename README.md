## Slime lazy src
### ดาวน์โหลด
```cmd
git clone https://github.com/kingslimes/slime-lazy-src.git
```
### CDN code
```url
https://kingslimes.github.io/docs/js/slime.lazy.min.js
```
### ตัวอย่างการใช้งาน
>**example:**
```html
<img slime-lazy-src="image.jpg" />
```
### สามารถนำไปใช้กับแทค
- img
- video
- embed
- iframe
