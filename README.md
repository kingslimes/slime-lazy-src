## Slime lazy page
### Download
```git
git clone https://github.com/kingslimes/slime-lazy-src.git
```
### Add attribute **slime-lazy-src**
>**example:**
```html
<img slime-lazy-src="image.jpg" />
```
### Support tag
- img
- video
- embed
- iframe
